# RedTurtleCakes
紅龜粿、草仔粿、壽桃

# 遊戲規則
```text
1.粿出現的時間 倒數完會減一顆❤ 給錯工具也會減一顆❤
2.蒼蠅出現是隨機出現的
3.中間的粿敲完換下一個 且會重新填滿倒數時間
```
